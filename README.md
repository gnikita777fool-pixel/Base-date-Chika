# Основные знания по базам данных SQL:

---

## Создание таблицы:

**CREATE TABLE** *<название таблиц>* |- создает таблицу

**CREATE TABLE** *<название таблиц>* ( |- *создание таблицы с переменными*
 <br> *<название переменной>* **INT** |- *создает переменную*
 <br> *<название переменной>* **VARCHAR(265)** ); |- *создает строчную переменную*

---

### Пример кода:

**CREATE TABLE** *student* (
 <br> *pr_name* **VARCHAR(265)**, 
 <br> *age* **INT**,
 <br> *tax* **INT**);
 
---

## Внесение данных в таблицу:

**INSERT INTO** *<название таблицы>* (*название переменной, ...,*) **Values** |- внесение данных в таблицу
<br> *('Maria',18,110)*, |- данные внесенные в переменные 
<br> *('Sveta',19,100),* 
<br> *('Olya',20,111);*

---

**P.S.** - главное вводить данные переменных по тому порядку по которые вы их расположили при наименовании переменных:
 <br> *(pr_name, age, tax),*
 <br> *('Maria',18,110);*

---

### Пример кода:

**INSERT INTO** *student* *(pr_name, age, tax)* **Values** 
<br> *('Maria',18,110),* 
<br> *('Sveta',19,100),* 
<br> *('Olya',20,111);*

---

## Вложенные запросы для SQL:

**select** *<столбец, ... ,>* * from *<таблица>*  |- выбор столбца из таблицы

 <br> * from |- пишеться после select, позволяет выбирать конкретные столбцы
 <br> **where** |- пишеться после select, позволяет выводить данные если оно совпадает с условием
 <br> **group by** |- позволяет группировать данные из таблиц
 <br> **heavy by** |- позволяет фильтровать данные из группировки
 <br> **order by** |- позволяет сортировать данные по стоблцу 
 <br> **limit** |- количестко значений которых надо вывести
 <br> **ask** |- сортировка по возрастанию 
 <br> **desk** |- сортировка данных по убыванию

---

### Пример кода:

<br> **select** para, SUM(money) as points
<br> **from** *student*
<br> **group by** para
<br> **order by** *points* **desc**
<br> **limit** 4;

---

## Условное соединение таблиц

**join ... on** |- соединение таблиц 
    <br> **right** |- сторона с которой начинается вывод
    <br> **left** |- сторона с которой начинается вывод
    <br> **cross** |- объединяет значения в пару

<br> **union** |- объединяет две таблицы выводя их в один отчет
<br> **all** |- делает так что бы одинаковые данные в отчете не дублировались 

---

## Условное изменение названия

<br> **AS** |- условное изменение названия таблицы, перменной
