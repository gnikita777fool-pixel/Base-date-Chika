# Основные знания по базам данных:

## Создание таблицы:

**CREATE TABLE** *<название таблиц>* - создает таблицу

**CREATE TABLE* *<название таблиц>* ( - *создание таблицы с переменными*
 <br> *<название переменной>* **INT** - *создает переменную*
 <br> *<название переменной>* **VARCHAR(265)**) - *создает строчную переменную*

### Пример кода:

**CREATE TABLE** *student* (
 <br> *pr_name* **VARCHAR(265)**, 
 <br> *age* **INT**,
 <br> *tax* **INT**);

## Внесение данных в таблицу:

**INSERT INTO** *<название таблицы>* (*название переменной, ...,*) **Values**

**INSERT INTO** student (pr_name, age, tax) **Values** 
<br> ('Maria',18,110), 
<br> ('Sveta',19,100), 
<br> ('Olya',20,111);

**select** - позволяет выбирать таблицы и данные из таблиц
 <br>  * from - пишеться после select, позволяет выбирать конкретные таблицы
 <br> **where** - пишеться после select, позволяет выводить данные если оно совпадает с условием

### Пример кода:

**select** *<таблица, ... ,>* * from *<столбец>*

 <br> grow by - позволяет группировать данные из таблиц
 <br> heavy by - позволяет фильтровать данные из группировки
 <br> order by - позволяет сортировать данные по стоблцу 
 <br> ask - сортировка по возрастанию 
 <br> desk - сортировка данных по убыванию
 <br> limit - количестко значений которых надо вывести

"*right, left or cross*" **join ... on** - соединение таблиц 
    <br> **right* - сторона с которой начинается вывод
    <br> **left** - сторона с которой начинается вывод
    <br> **cross** - объединяет значения в пару

AS - условное изменение названия таблицы
union - объединяет две таблицы выводя их в один отчет
     all - делает так что бы одинаковые данные в отчете не дублировались
